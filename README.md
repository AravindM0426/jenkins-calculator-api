# 🚀 Jenkins Calculator API

A simple Spring Boot Calculator REST API developed for learning **Git, GitHub, Maven, JUnit, Jenkins, and CI/CD**.

---

## 📌 Project Objective

This project was created to gain hands-on experience with:

- Spring Boot REST API Development
- Java Records
- JUnit 5 Testing
- Maven Build Automation
- Git Version Control
- GitHub Repository Management
- Jenkins Continuous Integration (CI)
- Jenkins Continuous Deployment (CD)

# 📡 Available APIs

| Method | Endpoint | Description |
|----------|----------|-------------|
| GET | /calculator/add | Addition |
| GET | /calculator/subtract | Subtraction |
| GET | /calculator/multiply | Multiplication |
| GET | /calculator/divide | Division |
| GET | /calculator/square | Square |
| GET | /calculator/sqrt | Square Root |
| GET | /calculator/factorial | Factorial |
| GET | /calculator/percentage | Percentage |

---

# 🧪 Unit Testing

The project contains JUnit 5 test cases covering:

- Addition
- Subtraction
- Multiplication
- Division
- Square
- Square Root
- Factorial
- Percentage
- Exception Scenarios
- Build Failure Demonstration (for Jenkins)

Execute tests using:

```bash
mvn test
```
